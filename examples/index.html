TW Select

Styled select box replacement, supporting the majority of the standard HTML interface for a select. The select box has full keyboard support matching native behaviour.

The select box also maintains a hidden form input with the given name to be compatible with HTML form posts.

Required
Configuration
<tw-select
  name="mySelect"
  placeholder="{{vm.select.empty}}"
  ng-model="vm.components.select"
  ng-required="{{vm.select.required}}"
  ng-disabled="{{vm.select.disabled}}"
  tw-options="[
    {'value': '', 'label': 'None selected'},
    {'value': '1', 'label': 'Option one'},
    {'value': '2', 'label': 'Option two'},
    {'value': '3', 'label': 'Option three'}
  ]">
</tw-select>

Date form control

Two way data binding via ng-model, with either a javascript Date object or an ISO8601 String date: 1994-11-05T08:15:30-05:00

Configuration
<tw-date
  ng-model="vm.components.date"
  ng-required="{{vm.date.required}}"
  ng-disabled="{{vm.date.disabled}}"
  tw-locale="{{vm.date.locale}}"
  min="1990-01-01"
  max="2015-12-31"
</tw-date>

Dynamic form control

The dynamic form control is a form control that can be fully configured based on a set of parameters.

This control can be used as the basis for a fully dynamic form based on an external specification.

Required
Must be at least {{vm.dynamic.minlength}} characters
Must not be longer than {{vm.dynamic.maxlength}} characters
Must be at least {{vm.dynamic.min}}
Must not be higher than {{vm.dynamic.max}}
Invalid characters
Must be a number
Configuration
<tw-dynamic-form-control
  type="{{vm.dynamic.type}}"
  placeholder="{{vm.dynamic.placeholder}}"
  options="{{vm.dynamic.options}}"
  ng-model="{{vm.model.dynamicFormControl}}"
  ng-required="{{vm.dynamic.required}}"
  ng-disabled="{{vm.dynamic.disabled}}"
  ng-minlength="{{vm.dynamic.minlength}}"
  ng-maxlength="{{vm.dynamic.maxlength}}"
  ng-pattern="{{vm.dynamic.pattern}}"
  ng-min="{{vm.dynamic.min}}"
  ng-max="{{vm.dynamic.max}}"
  tw-validation>
</tw-dynamic-form-control>
