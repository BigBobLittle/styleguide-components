TW Presentation Pattern

tw-presentation-pattern is a simple attribute based directive that formats text entered into controls. The model value is preserved without formatting, and validation continues to function on the unformated value.

<input type="text"
  class="form-control"
  ng-model="{{$ctrl.model}}"
  ng-minlength="5"
  ng-maxlength="9"
  ng-required="true"
  tw-presentation-pattern="{{$ctrl.pattern}}"
  tw-validation
  tw-focusable />
{{$ctrl.model | twPresentationPattern:$ctrl.pattern }} 
{{ "{{$ctrl.model}}" | twPresentationPattern : "{{$ctrl.pattern}}" }}
