TW Popover

tw-popover is an attribute based directive that enables a floating description to be revealed when an element is clicked or focused. It is configured through data attributes.

tw-popover requires bootstrap.js

Simple text - Custom template 
<a tw-popover
  title="Popover title"
  data-toggle="popover"
  data-placement="top"
  data-content="Full description copy, explaining in more detail"
  role="button"
  tabindex="0">
  Link text
</a>

<a tw-popover
  title="Injected into .popover-title"
  data-toggle="popover"
  data-placement="bottom"
  data-content="Injected into .popover-content"
  data-template="<div class='popover' role='tooltip'><div class='arrow'></div><h3 class='popover-title'></h3><div class='popover-content'></div><a href=''>Custom Link</a></div>"
  role="button"
  tabindex="0">
  Custom template
</a>
