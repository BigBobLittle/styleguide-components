Requirement form

The requirmeent form uses the dynamic form control to build a fully featured form. It accepts a model object which it permutes based on a set of requirements. The requirements object should match that returned by the TransferWise API

Edit configuration
Configuration
<tw-requirement-form
  requirements="$ctrl.requirements"
  model="{{$ctrl.model | json}}"
  on-refresh-requirements=""></tw-requirement-form>
