Date format filter

twDateFormat is an Angular.js custom filter to format dates. It is useful for internationalization since different countries follow during date formats (e.g. dd/MM/yyyy vs MM/dd/yyyy vs yyyy/MM/dd) and time formats (12h vs 24h). The twDateFormat filter takes 5 optional parameters:

  1. locale
  2. patternName, which can be one of the following values: LongDate, LongDateShortYear, LongDateNoYear, ShortDate, ShortDateLongYear and ShortDateNoYear
  3. hasWeekday (e.g. Monday)
  4. hasDayTime (e.g. 2:35 PM)
  5. hasSuffix (e.g. 21st)
